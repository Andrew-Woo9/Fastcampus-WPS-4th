# HTML Day4

### Atom sass-autocompile 패키지 설치

**Ubuntu**  
`sudo apt-get install -y nodejs`   
`sudo apt-get install -y npm`

**macOS**  
`brew install node`

-

**nodejs 및 npm설치 이후**  

터미널에서 `node -v` 입력

에러가 난다면  
`nodejs -v` 입력  

버전이 정상적으로 출력되면  
 `sudo ln -s /usr/bin/nodejs /usr/bin/node`입력  

이후 `node-sass` npm패키지 설치  
`sudo npm install node-sass -g`
